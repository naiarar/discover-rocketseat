Cascata

Temos 3 formas de adc um estilo css no html

  1. Inline: Ao escrever a sua tag coloca o style direto. Ex.: 
                        <h1 style="color:green">Titulo</h1>
  2. Tag style: coloca a tag no inicio do codigo html.Ex.: 
                        <style>
                        h1 {
                            color: gray;
                        }
                    </style>
  3. Link externo: O link é anexado a página primcipal depois do titulo.Ex.: 
                    <link> rel="stylesheet"href="style.css">

A leitura da cascata é de cima pra baixo, SEMPRE vai funcionar a de baixo se tiver em duplicidade

O selector universal * tem valor de 0, mesmo que eles esteja abaixo de todos ele só vai valer se não houver nenhum outro seletor especifico

    Nível de força do menos forte para o mais forte:
  • Selector Universal * é o mais fraco =0
  • Element type selector e pseudo element =1
  • Classes e attribute =10
  • ID selector =100
  • Inline é o mais forte =1000

    • As forças dos elementos podem ser somadas

Caso utilize uma biblioteca CSS de alguém e não conseguir editar algo, pode ser usado !important que ele força acima de tudo o css. Mas só deve ser usado NESTE caso.
