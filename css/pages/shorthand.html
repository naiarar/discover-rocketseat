Shorthand

É um método para colocar diversas propriedades do CSSS de maneira mais resumida e mais legivel

Exemplo background

Método CSS

            {
                background‐color: #000;
                background‐image: url(images/bg.gif);
                background‐repeat: no‐repeat;
                background‐position: left loop;
            }

Método Shorthand

            {
                background: #000 url(images/bg.gif) no‐repeat left loop;
            }

Exemplo font

Método CSS

            {
                font‐style: italic;
                font‐weight: bold;
                font‐size: .8em;
                line‐height:1.2;
                font‐family: Arial, sans–serif;
            }

Método Shorthand

            {
                font: italic bold .8em/1.2 Arial, sans–serif;
            }

Importante

  • Não irá considerar propriedades anteriores, o shorthand é mais forte que as propriedades normais
  • valores não especificados irão assumir valor padrão
  • geralmente a ordem da escrita não importa, mas, se houver muitas propriedades com valores semelhantes, poderemos encontrar problemas
